### 语言占比环形图
- 定义：本项目的核心产出物——根据用户公开仓库的语言字节统计生成的 SVG 环形图，用于展示代码仓库中各编程语言的占比分布。图表支持浅色/深色主题自适应、最多 9 种语言单独显示、第 10 项汇总为 Other，并具备小占比扇段最低可见角度等可视化增强。
- 别名：language donut chart、donut chart

### 主页仓库
- 定义：GitHub 个人主页 README 所在的仓库，仓库名与用户名同名（如 `octocat/octocat`）。本项目默认自动排除该仓库不参与语言统计，且生成结果会提交回此仓库。
- 别名：profile repository

### 版本化文件名
- 定义：为避免 GitHub 对同名图片缓存导致 README 无法刷新最新图表，每次生成时基于内容摘要生成带哈希后缀的 SVG 文件名（如 `language-donut-a1b2c3d4e5f6.svg`），旧版本文件会被自动清理。
- 别名：versioned filename

### Other
- 定义：当语言数量超过配置的最大命名语言数（默认 9）时，剩余所有语言被聚合为一项名为 Other 的汇总项，其颜色可在配置中自定义，默认使用中性灰色。

### 代码推送通知
- 定义：通过在各个代码仓库部署 notify-profile.yml 工作流，监听 push 事件后向个人主页仓库发送类型为 code-pushed 的 repository_dispatch 事件，从而触发主页语言图表的自动更新。需要配置仅授权主页仓库的 fine-grained PAT 作为 PROFILE_REPO_TOKEN。
- 别名：notify-profile、cross-repo notification
