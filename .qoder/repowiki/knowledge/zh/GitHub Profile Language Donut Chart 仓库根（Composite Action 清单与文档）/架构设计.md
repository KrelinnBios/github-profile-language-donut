本模块为仓库根目录，不包含业务代码，仅承载 Composite Action 的对外契约与项目元信息：
- `action.yml` 是唯一的 Action 入口，声明 `runs.using: composite` + `shell: bash`，将 5 个输入参数映射为对 `src/generate.py` 的命令行调用，并通过 `steps.generate.outputs.*` 透传 `image`、`changed` 两个输出。
- `.github/workflows/test.yml` 在 `ubuntu-latest` 上通过 `python -m unittest discover -s tests -v` 运行单元测试，作为该 Action 的 CI 门禁。
- 三语 README（`README.md` / `README.zh-TW.md` / `README.en.md`）位于根目录，顶部互相链接；示例工作流与配置文件由 README 引用至 `examples/` 子目录供使用者复制。
- `.gitignore`、`.gitattributes`、`LICENSE` 提供版本控制与许可元数据。
依赖方向单向：外部用户 → `action.yml` → `src/generate.py` → `src/language_donut/*`，本根目录不引入任何运行时依赖。
