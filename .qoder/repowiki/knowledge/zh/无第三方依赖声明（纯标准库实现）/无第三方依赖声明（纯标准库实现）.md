该仓库是一个 GitHub Action，其 Python 代码完全基于 Python 标准库实现，未引入任何第三方依赖。所有导入均为内置模块：`argparse`、`pathlib`、`json`、`os`、`urllib.request`、`html`、`math`、`colorsys`、`hashlib`、`re`、`collections.Counter` 等。仓库中不存在 `requirements.txt`、`pyproject.toml`、`setup.py`、`Pipfile`、`poetry.lock`、`go.mod`、`package.json` 或 `vendor/` 目录等任何依赖清单或锁定文件。Action 通过 `action.yml` 以 composite runner 方式直接执行 `src/generate.py`，运行环境依赖由 GitHub Actions 提供的 Python 运行时提供，无需额外安装依赖。因此本仓库不涉及第三方依赖管理策略。
