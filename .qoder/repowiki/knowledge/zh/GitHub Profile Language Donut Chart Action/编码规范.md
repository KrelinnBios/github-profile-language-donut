- Action 输入/输出统一在 `action.yml` 的 `inputs:` / `outputs:` 块中以键值对声明，并通过 `${{ inputs.* }}` / `${{ steps.generate.outputs.* }}` 引用。
- 配置加载采用「默认常量 + JSON 覆盖」模式：先拷贝 `DEFAULT_*` 字典，再用 `raw.get(key, default)` 逐项覆盖，同时保留旧键名到新键名的兼容映射。
- GitHub API 请求集中在 `github_json()` 单点封装，统一注入 `Accept`、`X-GitHub-Api-Version`、`Authorization` 头，并将 HTTPError 包装为带详细信息的 `RuntimeError`。
- README 提供多语言版本（简体中文、繁体中文、English），各语言文件通过顶部导航互相链接。
- 示例工作流与配置文件放在 `examples/` 目录下，使用者按说明复制到目标仓库对应路径即可复用。
